# ШУМ — простий лендінг

Структура проєкту:
- `index.html` — уся структура сторінки
- `styles.css` — стилі
- `script.js` — меню, підсвітка активного пункту, горизонтальний скрол у блоці проєктів
- `assets/images/` — усі зображення-заглушки

## Як замінити картинки
Усі сірі блоки — це звичайні `img` всередині `.image-box`.
Достатньо замінити файл у `assets/images/` або підставити свій шлях у `src`.

## Як додати новий проєкт
Скопіюйте ще один `<article class="project-card">...</article>` всередині блоку `.projects-track`.

## Як змінити контакти або команду
Відредагуйте відповідні секції в `index.html`.

## Горизонтальний скрол
Горизонтально прокручується лише секція `Наші проєкти`.
На комп'ютері вона рухається колесом миші, на телефоні — свайпом.
