Проєкт ШУМ

Структура:
- index.html
- script.js
- css/reset.css
- css/header.css
- css/about.css
- css/projects.css
- css/team.css
- css/contacts.css
- css/responsive.css
- assets/images/*

Що змінено:
- CSS розбитий на окремі файли по екранах/блоках
- прибрані CSS-змінні
- селектори розділені, без довгих списків через кому
- десктоп підлаштований під розміри 1920x1080 за наданими відступами
- горизонтальний скрол залишений тільки в блоці проєктів
- на мобільному проєкти видно як свайп-картки з натяком на наступну

Як швидко міняти контент:
- тексти редагуються в index.html
- розміри й кольори хедера: css/header.css
- блок О нас: css/about.css
- блок Наші проєкти: css/projects.css
- блок Команда: css/team.css
- блок Контакти: css/contacts.css
- адаптив: css/responsive.css


Швидка зміна кольорів:
- усі основні кольори винесені в `css/reset.css` у блок `:root`
- щоб швидко змінити палітру сайту, редагуй змінні `--color-*`
- головні змінні: фон сторінки, текст, хедер, картки проєктів, CTA-картка, сірі фото-заглушки
