Проєкт ШУМ

Поточна фінальна версія на цей момент.

Структура:
- index.html
- script.js
- css/reset.css
- css/header.css
- css/about.css
- css/projects.css
- css/team.css
- css/contacts.css
- css/responsive.css
- assets/images/*

Що є в цій версії:
- CSS розбитий на окремі файли по блоках
- без CSS-змінних
- без довгих складних селекторів через кому
- хедер, блоки та адаптив винесені окремо
- блок "Наші проєкти" лишився горизонтальним
- для блоку проєктів додана підтримка горизонтального скролу колесом миші через JavaScript
- на мобільних і тачпристроях блок проєктів працює як нативний горизонтальний свайп/скрол
- бургер-меню, плавний перехід по якорях і підсвітка активного пункту меню теж у script.js

Як редагувати:
- усі тексти і посилання: index.html
- хедер: css/header.css
- блок "О нас": css/about.css
- блок "Наші проєкти": css/projects.css
- блок "Команда": css/team.css
- блок "Контакти": css/contacts.css
- адаптив: css/responsive.css
- логіка меню, якорів і горизонтального скролу проєктів: script.js

Як додавати новий проєкт:
- у index.html знайти блок <div class="projects-track">
- скопіювати цілий <article class="project-card"> ... </article>
- вставити його після попередньої картки
- замінити назву, текст, учасників і картинки

Картинки:
- сірі або тестові зображення лежать у assets/images/
- їх можна просто замінити своїми файлами з тими ж назвами
- або змінити шляхи до зображень у index.html

Нотатки:
- у коді місцями залишені закоментовані заголовки/тексти, які можна повернути або остаточно видалити
- нижні великі паддінги деяких секцій поки лишені як є, бо це окремо обговорювалось
- якщо далі буде новий дизайн або точніші відступи, найкраще правити окремо десктоп і окремо responsive.css
